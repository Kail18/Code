# FitLog

## This app is for logging and building workout programs.

## This app is using React + Typescript as its frontend

### How to set up the environment for VSCode

- <p First you need to install node>
- <p Second you need to install typescript>
- <p Third you need VSCode>
- <p In VSCode navigate to the terminal till your in the project directory>
- <p run: npm i (this will install the npm dependencies)>
- <p next run: npm run dev (this will run the server as a local host)>
- <p copy and paste the http localhost into your browser to see the react app>
